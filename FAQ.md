# ❓ Frequently Asked Questions (FAQ)

- **Q: [Why can't I see the export buttons in Copilot?](ca://s?q=Why_can%27t_I_see_the_export_buttons_in_Copilot)**  
  A: Reload the page. Sometimes Copilot takes a moment to initialize and the buttons don’t appear.

- **Q: [What happens if I click the export button and nothing happens?](ca://s?q=Export_button_not_working)**  
  A: Check the popup message. It shows information about the error.

- **Q: [If an error occurs during export, is it deducted from the export limit?](ca://s?q=Export_limit_error_behavior)**  
  A: No. The limit only decreases when messages are successfully exported.

- **Q: [Is there any limit on exports or messages?](ca://s?q=Export_and_message_limits)**  
  A: Currently there is only a free plan with 10 weekly exports and 50 messages per export.

- **Q: [When does the export limit reset?](ca://s?q=Export_limit_reset_time)**  
  A: It resets one week after the last reset.

- **Q: [Can I change the extension theme?](ca://s?q=Change_extension_theme)**  
  A: Yes. There is a button in the popup header to switch the theme.

- **Q: [Can I change the extension language?](ca://s?q=Change_extension_language)**  
  A: Yes. There is a button in the popup header to change the language.

- **Q: [Can the extension be used in other browsers?](ca://s?q=Use_extension_in_other_browsers)**  
  A: The extension is tested in Chrome and may work in other Chromium‑based browsers.

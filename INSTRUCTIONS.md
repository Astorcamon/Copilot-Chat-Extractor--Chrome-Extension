## 🔧 Installation

Here are two ways to install this extension.

### Automated (Recommended)

Install the extension from **Chrome Web Store** → <!--CCE-Extension--> **(coming soon)**

### Manual Installation (Release Package)

Install the extension using _Chrome’s developer mode_:

1. Download the latest **CCE-Extension.zip** → [Releases](../../releases).
2. Unzip the package to some place in your computer.
3. On your browser, put this as url `chrome://extensions`.
4. Turn on **Developer mode** (toggle on top-right corner).
5. Click on **Load unpacked** button (top-left corner).
6. Select the folder you unzipped.

### Tips

> 💡 After install, don´t forget to pin the extension on the browser.
>
> - Click extensions button (puzzle button upper-right corner) and pin CCE.

## 📝 How to export conversations

The extension has two ways to export **Copilot conversations**.

### Export from Popup

1. Click the (<img src="./.resources/CCE128.png" alt="CCE" width="16px" align="center"/>) icon in your browser (top-right corner).
2. Open **Copilot.com** (popup, tab, or side panel).
3. Open a conversation (with messages).
4. Click **MD** or **DOCX**.
5. Select where to save the document.

### Export from Copilot

1. Open **Copilot.com** (popup, tab, or side panel).
2. Open a conversation (with messages).
3. Open ( <img src="./.resources/CCE128.png" alt="CCE" width="16px" align="center"/>) **CCE - Menu** (inside Copilot).
4. Click **Export - MD** or **Export - Docx**.
5. Select where to save the document.

## 📌 Bonus - Pin CCE to Desktop, Taskbar or Start Menu (Windows only)

Add **Copilot Chat Extractor** to **Windows desktop** for quick access.

### The easy way (recommended)

1. Download and install → [Chrome](https://www.google.com/intl/en/chrome/).
2. Download and unzip → [CCE-Desktop.zip](./.resources/CCE-Desktop.zip).
3. Follow the instructions to **Customize the shortcut icon** below.

### Manual shortcut creation

If you want to create the desktop shortcut manually:

1. Right-click inside the desktop or any folder and select `"New > Shortcut"`.
2. Right-click the shortcut and select `"Properties"`.
3. Add this in the **Target** field:  
   `%PROGRAMFILES%\Google\Chrome\Application\chrome.exe --app=https://copilot.microsoft.com`
4. Add this in the **Open in** field:  
   `%PROGRAMFILES%\Google\Chrome\Application`
5. Follow the instructions to **Customize the shortcut icon** below.

> **Note**: the path for `chrome.exe` may vary on different environments.

### Customize the shortcut icon

If the downloaded shortcut lost the (<img src="./.resources/CCE128.png" alt="CCE" width="16px" align="center"/>) [CCE icon](./.resources/CCE.ico):

1. Right click on the shortcut and select `"Properties"`.
2. Click on **Change icon**.
3. Search and select (<img src="./.resources/CCE128.png" alt="CCE" width="16px" align="center"/>) [CCE.ico](./.resources/CCE.ico) then accept changes.

### Use the shortcut

Use the shortcut like any Windows shortcut:

- Move to any folder included Desktop
- Right-click on the shortcut and select `"Pin to Start Menu"`.
- Drag and drop to taskbar.
